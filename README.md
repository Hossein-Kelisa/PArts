# PArts

A modern full-stack web application for a professional photography and videography studio.

The goal of **PArts** is to provide an elegant online presence where visitors can explore photo and video portfolios, learn about the studio, and submit booking requests. The project also includes a secure administration panel for managing website content and customer requests.

> **Project Status:** 🚧 In Development

---

## 🎯 Project Goals

- Build a modern, responsive, and user-friendly website.
- Showcase photography and videography work professionally.
- Allow visitors to submit booking requests online.
- Provide an admin dashboard for content management.
- Apply full-stack development best practices.
- Deploy the application to a production environment.
